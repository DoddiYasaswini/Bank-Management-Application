## 🚀 Overview

The **Bank Management Application** is a menu-driven console application developed using **C++** that simulates core banking operations.

This project allows users to:

✅ Create Bank Accounts

✅ Deposit Money

✅ Withdraw Money

✅ Check Account Balance

✅ Display Customer Records

The system uses **Object-Oriented Programming (OOP)** and **File Handling** to manage customer data securely and permanently.

---

# ✨ Features

## 🆕 Create New Account

Create and store customer bank account details.

## 💰 Deposit Amount

Deposit money into an existing account.

## 💸 Withdraw Amount

Withdraw money securely with balance validation.

## 📊 Balance Inquiry

Check available account balance instantly.

## 📋 Display All Accounts

View all customer account records in formatted form.

## 💾 Persistent Storage

All records are stored using:

```text
bank.txt
```

This ensures customer data remains saved even after closing the application.
